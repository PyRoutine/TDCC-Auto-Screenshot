# 📸 股東e票通 - 獨立補圖專用小幫手

這是一套基於 Python 與 Selenium 開發的 RPA（機器人流程自動化）工具，專為需要頻繁登入「股東e票通」網站查詢與截圖的使用者所設計。透過直覺的圖形化介面（GUI），協助使用者一鍵批次完成多帳號、多檔股票的投票紀錄截圖，大幅節省人工作業時間。

## ✨ 主要功能

* **多帳號與憑證管理**：支援綁定多組身分證與對應名稱，並可針對不同帳號個別設定登入方式（支援：券商網路下單憑證、自然人憑證、行動自然人憑證）。
* **全自動批次截圖**：只需貼上股票代號（支援 Excel 欄位複製貼上），程式即可自動切換帳號、搜尋代號、進入明細並完成截圖。
* **智慧防呆與略過機制**：
    * 自動偵測網頁實際戶名，避免檔名錯誤。
    * 若偵測到該檔股票狀態為「未投票」將自動跳過。
    * 智慧比對本地端資料夾，若圖片已存在則自動跳過，避免重複作業。
* **彈性設定**：可自由調整「截圖等待速度」（應對不同網路環境），並支援「各帳號獨立資料夾」或「合併存檔」兩種檔案管理模式。
* **抽獎活動捕捉**：可設定遇到平台抽獎畫面時，自動暫停 5 分鐘讓使用者手動參與。
* **詳盡的執行報告**：每次任務結束後，自動於 `Log` 資料夾生成純文字報告，紀錄成功與失敗（或跳過）的明細。

## 🚀 快速上手 (一般使用者)

如果你不需要修改程式碼，只想直接使用本工具，請前往 [Releases](../../releases) 頁面下載最新版的 `.exe` 執行檔。

### 準備工作
1. 本程式核心依賴微軟 Edge 瀏覽器進行自動化，請確保你的電腦已安裝 **Microsoft Edge**。
2. 準備好你的憑證環境（如：需讀卡機與實體卡片，或確保已安裝券商下單憑證）。

### 使用步驟
1. 執行 `補圖.exe`（首次開啟可能需要等待幾秒鐘載入）。
2. 切換至 **[設定]** 標籤頁，新增你的帳號資訊（姓名、身分證、登入憑證類型）。
3. 切換回 **[補圖任務]** 標籤頁，勾選本次要執行的帳號。
4. 在「股票代號」區塊貼上你需要截圖的股票代號（可多組，用換行或逗號分隔）。
5. 點擊 **「啟動補圖任務」**，放開雙手讓程式自動執行！
6. 任務完成後，系統會自動跳出提示並為你打開截圖存放的資料夾。
