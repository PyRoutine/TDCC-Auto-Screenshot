# 📸 股東e票通 - 全能自動化小幫手 (TDCC AutoTool)

這是一套基於 Python 與 Selenium 開發的 RPA（機器人流程自動化）工具，專為需要頻繁登入「股東e票通」網站的使用者所設計。透過直覺的圖形化介面（GUI），不僅能協助使用者一鍵批次完成多帳號的投票紀錄截圖，更具備 **E-Gift (電子紀念品) 資格掃描與自動彙整** 功能，大幅節省繁瑣的人工作業時間。

## ✨ 主要功能

* **🎁 E-Gift 資格全面掃描 (全新功能)**：自動遍歷指定帳號的投票紀錄，透過網頁解析精準抓出所有符合「E-Gift」領取資格的股票與開始領取日期，並自動彙整匯出成 `E-Gift領取清單.txt`。
* **🔐 多帳號與憑證管理**：支援綁定多組身分證與對應名稱，可針對不同帳號個別設定登入方式（支援：券商網路下單憑證、自然人憑證、行動自然人憑證）。**支援 Excel 批次匯入帳號資料**。
* **📸 全自動批次截圖**：只需貼上股票代號（支援 Excel 欄位複製貼上），程式即可自動切換帳號、搜尋代號、進入明細並完成截圖。
* **🛡️ 智慧防呆與略過機制**：
    * 自動偵測網頁實際戶名，避免檔名與實際帳號不符。
    * 支援 E-Gift 判斷：若偵測到該檔股票符合 E-Gift 資格可免截圖，將自動跳過。
    * 狀態判定：若偵測到該檔股票狀態為「未投票」將自動跳過。
    * 智慧比對本地端資料夾，若圖片已存在則自動跳過，避免重複作業。
* **⚙️ 彈性設定**：可自由調整「截圖等待速度」（應對不同網路環境），並支援「各帳號獨立資料夾」或「合併存檔 (檔名標註帳號)」兩種檔案管理模式。
* **🎉 抽獎活動捕捉**：可設定遇到平台抽獎畫面時，自動暫停 5 分鐘讓使用者手動參與。
* **📊 詳盡的執行報告**：每次任務結束後，自動於 `Log` 資料夾生成純文字報告，紀錄成功與失敗（或跳過）的明細。

## 🚀 快速上手 (一般使用者)

如果你不需要修改程式碼，只想直接使用本工具，請前往 [Releases](../../releases) 頁面下載最新版的 `.exe` 執行檔。

### 準備工作
1. 本程式核心依賴微軟 Edge 瀏覽器進行自動化，請確保你的電腦已安裝 **Microsoft Edge**。
2. 準備好你的憑證環境（如：需讀卡機與實體卡片，或確保已安裝券商下單憑證）。

### 使用步驟
1. 執行下載的 `.exe` 程式（首次開啟可能需要等待幾秒鐘載入）。
2. **【帳號建檔】** 切換至 **[設定]** 標籤頁，新增你的帳號資訊，或點擊「產生匯入範本」後使用 Excel 批次匯入。
3. **【執行補圖】** 切換至 **[補圖任務]** 標籤頁，勾選要執行的帳號，貼上股票代號並點擊「啟動補圖任務」。
4. **【掃描 E-Gift】** 切換至 **[E-Gift掃描]** 標籤頁，勾選要查詢的帳號並點選執行，程式會自動幫你整理好領取清單！
5. 任務完成後，系統會自動跳出提示，並為你打開對應的資料夾或報告檔。

## 🛠️ 開發與建置環境 (開發者)

若你想自行修改原始碼或重新打包，請參考以下環境配置：

### 環境依賴
* Python 3.8+
* Windows 作業系統 (因涉及硬體金鑰加密與特定 Windows API 視窗控制)

### 依賴套件 (requirements.txt)
```text
selenium
cryptography
Pillow
pandas
openpyxl
```

### 打包指令 (PyInstaller)
若要將修改後的 Python 腳本打包為執行檔，可使用本專案隨附的 `build_release.py` 自動化腳本，或手動執行：
```bash
pyinstaller --noconsole --onefile --name=TDCC-AutoTool --icon=icon.ico TDCC-AutoTool.py
```
